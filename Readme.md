Blog API
========

A RESTful backend API for a blog platform built with **Node.js, Express, and MongoDB**.This project implements authentication, protected routes, and full CRUD operations for blog posts.

🚀 Features
-----------

*   User authentication with **JWT**
    
*   **Password hashing** using bcrypt
    
*   Protected routes using **authentication middleware**
    
*   Full **CRUD operations for blog posts**
    
*   **Ownership validation** (only the post author can edit or delete)
    
*   MongoDB database using **Mongoose**
    
*   Structured backend architecture (controllers, routes, models, middleware)
    

🛠 Tech Stack
-------------

*   Node.js
    
*   Express.js
    
*   MongoDB
    
*   Mongoose
    
*   JSON Web Token (JWT)
    
*   bcrypt

🔐 Authentication Flow
----------------------

1.  User signs up with name, email, and password.
    
2.  Password is **hashed using bcrypt** before storing in MongoDB.
    
3.  On login, credentials are verified.
    
4.  A **JWT token** is generated and returned.
    
5.  Client sends the token in the header:
